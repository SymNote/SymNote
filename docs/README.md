## 🚀 Getting Started

Follow these steps to run the documentation locally.

### 1. Install dependencies

Navigate to the `docs` folder and run:

```sh
npm install
```

### 2. Start the development server
```sh
npm run dev
```

Then open your browser at:

```
http://localhost:4321/SymNote
```
